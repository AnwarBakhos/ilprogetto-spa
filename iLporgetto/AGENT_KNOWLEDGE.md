## 2. TECH STACK

**Framework:** Vite 8 + React 19 (NOT Next.js — never suggest Next.js migration)

- Files in `api/` run in Vercel's Node.js runtime, COMPLETELY SEPARATE from Vite
- NEVER import from `src/` in any `api/` file — path aliases don't resolve

- Keep all `api/` files self-contained with data inlined

## 11. CATALOG PAGE — HOW IT WORKS

**Filter pills (desktop sticky bar + mobile horizontal scroll):**
- Click pill → `handlePillClick(id)` → scrolls to `document.getElementById('card-{id}')`
- Offset: nav height (76px) + sticky bar height (measured live) + 16px gap
- Does NOT open the drawer — scroll only

**Product card click:**
- `handleCardClick(id)` → toggles `activeId` state → updates URL silently
- `navigate({ search: { product: id }, replace: true, resetScroll: false })`
- `resetScroll: false` is CRITICAL — prevents TanStack from jumping to top of page
- No scroll on card click — drawer opens in place

**Desktop drawer:**
- Injects after the row containing the active product (rows of 3)
- Fixed height 480px — 3 columns: photo | copy + features | mini booking form
- `borderTop: '2px solid var(--sand)'` connects visually to active card's sand bottom bar
- Active card gets `outline: 2px solid var(--sand)` and `3px sand bottom bar`

**Mobile drawer:**
- Vertical accordion below each card
- Mobile filter strip is horizontal-scrollable (not sticky)

**Mini booking form (in drawer):**
- 3 fields: First Name, Email (required for Resend), Phone
- Auto-picks next available date from `getAvailableDates()`
- Posts to `/api/book`

## 12. BOOKING FLOW

1. User visits `/booking` or `/booking?service={productId}`
2. `BookingCalendar` component: step 1 date → step 2 time → step 3 form
3. Submit → POST `/api/book`
4. On success → `window.location.href = '/booking/confirmed?id=...&date=...&time=...&email=...&name=...'`
5. `/booking/confirmed` reads params via `new URLSearchParams(window.location.search)`
6. Shows confirmation with booking ID, next steps grid, contact links

## 13. GLOBAL COMPONENTS DETAIL

### ServiceDrawer
- **Desktop:** vertical tab on right edge (`position: fixed, top: 50%, right: 0`)
  - Tab slides left when open (right: 360px → 0px transition)
  - Panel slides in from right (translateX 100% → 0)
- **Mobile:** bottom bar (52px) + bottom sheet (72vh) slides up
- **3 accordions:** Service Areas | 10% Discount form | Contact
- **Outside click:** `drawerRef` (desktop) + `mobileRef` (mobile) — both checked
- **Discount form:** POSTs to `/api/newsletter`

### ExitIntentPopup
- **Triggers:** 30 seconds on page OR 50% scroll depth (whichever first)
- **Suppressed:** `localStorage.setItem('ilp_popup_dismissed', '1')` after close
- **Form:** POSTs to `/api/newsletter`
- **Never shows again** after dismissal or form submission

### CookieConsent
- **Appears:** first visit only (`localStorage.getItem('ilp_cookie_consent')`)
- **Accept/Decline:** both store to localStorage and hide the banner
- **Never blocks** other UI elements (z-index: 950, pointer-events-auto)
